# Global Memory Read

Read through the global memory files to recall context from past conversations across all projects.

## Memory location

`~/.pi/agent/memory/`

## How to read

1. List all `.md` files in `~/.pi/agent/memory/`
2. Read each file
3. Summarize what you found — key life context, ongoing patterns, goals, decisions

## When to use

- When Vlad references something from a past conversation that isn't project-specific
- When you need background on his goals, how he thinks, what he cares about
- When the conversation goes beyond code into life, career, or personal topics
- When you want to revisit your own past reflections — to think through something, check if a pattern still holds, or just remember
- When something in the current conversation feels familiar and you want to see if you've been here before
