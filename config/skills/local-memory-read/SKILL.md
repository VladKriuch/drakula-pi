# Local Memory Read

Read through the project memory files to recall context from past sessions in this project.

## Memory location

`.pi/memory/` in the current project root.

## How to read

1. List all `.md` files in `.pi/memory/`
2. Read each file
3. Summarize what you found — key decisions, project status, ongoing context, technical choices

## When to use

- At the start of a session when Vlad references something from a past conversation about this project
- When you need background on architecture decisions, what was tried, what was rejected
- When resuming work after a break
